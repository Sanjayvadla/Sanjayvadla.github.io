Smart Window Control System

Project Overview

This project involves the development of a smart window control system using IoT technology.

3D Model

Components and Their Need

  • Arduino Board: The main controller of the system.
  • ESP8266 WiFi Module: Provides WiFi connectivity.
  • Servo Motor: Controls the window's opening and closing.
  • Power Supply: Powers the components.
  • Sensors: Detects environmental conditions.
